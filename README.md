# Personal Finance Frontend

React frontend for the Personal Finance Tracker API.

## Tech Stack
- React 18
- JavaScript (ES6+)
- Fetch API for HTTP requests
- JWT authentication via localStorage

## Features
- Login with JWT token
- List personal transactions
- Filter by user (each user sees only their own data)

## How to Run

\`\`\`bash
npm install
npm start
\`\`\`

The app will be available at `http://localhost:3000`.

> Requires the [Personal Finance API](https://github.com/junioriankoski/personal-finance) running on `http://localhost:8080`.
